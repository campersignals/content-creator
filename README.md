# Content Generator System

A powerful multi-platform content generation tool powered by AI.

## Features
- Generate content for Blog, Instagram, TikTok, and YouTube Community.
- Author profiles with style analysis.
- Image generation with Imagen 4.0.
- Vercel Postgres integration.

## Setup
1. Configure Environment Variables (`DATABASE_URL`, `GOOGLE_API_KEY`).
2. Run `npm install`.
3. Run `npm run dev`.
